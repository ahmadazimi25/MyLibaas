# MyLibaas Mobile App

This is the React Native mobile application for MyLibaas.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run on Android:
   ```bash
   npx react-native run-android
   ```
   Or on iOS:
   ```bash
   npx react-native run-ios
   ```

You may need to set up Android Studio or Xcode for mobile development.
